# AutoLoop Public Export

This is a sanitized local export of AutoLoop for publication review. It contains the reusable toolkit, templates, prompts, tests, generic docs, and sanitized verification fixtures.

This export intentionally excludes private repository history, raw internal work orders, raw worker reports, raw trial notes, named real-project case studies, local review inputs, target-project data, credentials, generated runtime artifacts, and deployment artifacts.
